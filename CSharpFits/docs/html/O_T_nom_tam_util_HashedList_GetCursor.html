HashedList.GetCursor Method
See Also Example

Return a Cursor over the entire list. The Cursor may be used to delete entries as well as to retrieve existing entries. A knowledgeable user can cast this to a HashedListCursor and use it to add as well as delete entries. NOTE: Cursor is initialized with -1 in case the start index is not specified.
