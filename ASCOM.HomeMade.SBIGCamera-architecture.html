ASCOM.HomeMade.SBIGCamera — Architecture

Homebrew ASCOM driver for SBIG USB cameras — .NET Framework 4.8 / .NET Standard 2.0

ASCOM Drivers (4)

  • • SBIGImagingCamera — main imager (STF/STT)
  • • SBIGGuidingCamera — STT/STX guiding sensor
  • • SBIGExternalCamera — STX remote head
  • • SBIGFW (Filter Wheel) — CFW-8

Driver Stack

  • • SBIGCamera — core abstraction layer
  • • SBIGClient — client library + image acquisition
  • • SBIGHub — COM local server, manages camera lifecycle
  • • SBIGCommon — request/response protocol, handlers

Platform Details

  • • Target: .NET Framework 4.8 (drivers) / .NET Standard 2.0 (SbigSharp)
  • • Primary target: STF-8300M + FW-8
  • • Native: HomeMade.SBIGUDrv.dll (x86/x64)
  • • SbigSharp by eliotg (forked)
